Foundations of Deep Learning for the Social Sciences

Overview

Deep learning (DL) has revolutionized how complex processes are modeled in fields including computer vision, natural language processing, computational biology, weather forecasting, and game playing, but is rarely used to model complex behavioral processes in the social sciences.

Prerequisites

Necessary prerequisites:
  • Working familiarity with generalized linear models, particularly linear regression and logistic regression
  • Basic proficiency in statistical programming (i.e., manipulating and analyzing data using a programming language like R or Python)
Helpful (but not necessary) prerequisites:
  • Linear algrebra
  • Multivariable calculus
