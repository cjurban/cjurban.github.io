Foundations of Deep Learning for the Social Sciences

Overview

Deep learning has revolutionized how complex processes are modeled in fields including computer vision, natural language processing, computational biology, weather forecasting, and game playing. Despite these impressive breakthroughs, and despite the potential benefits, deep learning is rarely used to model complex behavioral processes in the social sciences.

During the first day of this workshop, you will learn how to build, fit, and evaluate deep learning models to predict behavioral outcomes. We will start by discussing fundamental deep learning concepts in the context of linear regression, then progress to the .

During the second day, you will learn about how traditional latent variable models used in the social sciences can be enhanced using deep learning.

Prerequisites

Necessary prerequisites:
  • Working familiarity with generalized linear models, particularly linear regression and logistic regression
  • Basic proficiency in statistical programming (i.e., manipulating and analyzing data using a programming language like R or Python)
Helpful (but not strictly necessary) prerequisites:
  • Linear algrebra
  • Multivariable calculus
  • Familiarity with basic machine learning concepts (e.g., hyperparameters, cross-validation)
